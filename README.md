# Not Real Life
## Tester Quick Start (Expo)

Poniżej są tylko kroki potrzebne do uruchomienia aplikacji po pobraniu repozytorium.

### 1. Wymagania
- Node.js 18+ (zalecane LTS)
- `npm`
- Aplikacja `Expo Go` na telefonie (Android/iOS) lub emulator/symulator

### 2. Instalacja
```bash
npm install
```

### 3. Start projektu
```bash
npx expo start
```

### 4. Uruchomienie aplikacji
- Telefon: zeskanuj kod QR w terminalu/aplikacji Expo.
- Android emulator: naciśnij `a` w terminalu.
- iOS simulator (macOS): naciśnij `i` w terminalu.
- Web: naciśnij `w` w terminalu.

### 5. Gdy coś się wysypie (cache)
```bash
npx expo start -c
```
